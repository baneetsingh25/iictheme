# IIC College Society Frontend

React + TypeScript + Vite starter for the college society website and admin panel.

## Stack

- React
- TypeScript
- Vite
- React Router
- Tailwind CSS
- React Hook Form
- Zod
- Axios

## Getting started

1. Install dependencies:
   `npm install`
2. Copy environment values:
   `Copy-Item .env.example .env`
3. Run the app:
   `npm run dev`

## Team boundaries

- Frontend team owns routes, components, forms, layout, UX states, and API client integration.
- Backend team owns APIs, auth, business logic, email workflows, QR generation, exports, and file processing.
- Supabase team owns schema, storage, auth policy support, and operational setup.
